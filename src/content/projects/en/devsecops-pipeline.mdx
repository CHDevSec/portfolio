## What

Built the security gates that run on every pull request:

- **SAST** with Snyk Code + Checkmarx — application logic flaws
- **SCA** with Snyk Open Source — third-party dependency CVEs
- **IaC scanning** with Kics + Polaris — Terraform/Helm/Kubernetes misconfigs
- **Secrets scanning** — pre-commit + CI safety net
- **Policy enforcement** — codified rules for what blocks vs warns

## How it stayed adopted

- Block only on **critical + reachable** findings — never on theoretical noise
- Findings annotate the PR with line-level context, not a separate dashboard
- Auto-bump dependency PRs for fixable CVEs
- Weekly metrics shared with engineering leads — "what's red, what's green"

## Outcome

Security became a property the engineering team owned, not a quarterly bottleneck.
