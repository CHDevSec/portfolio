## O quê

Construí os gates de segurança que rodam em cada pull request:

- **SAST** com Snyk Code + Checkmarx — falhas de lógica de aplicação
- **SCA** com Snyk Open Source — CVEs em dependências de terceiros
- **IaC scanning** com Kics + Polaris — misconfigs em Terraform/Helm/Kubernetes
- **Secrets scanning** — pre-commit + safety net no CI
- **Policy enforcement** — regras codificadas pro que bloqueia vs alerta

## Como sobreviveu adoption

- Bloquear só em **critical + reachable** — nunca em ruído teórico
- Findings anotam o PR com contexto de linha, não dashboard separado
- Auto-bump de PRs de dependências pra CVEs com fix
- Métricas semanais com tech leads — "o que tá vermelho, o que tá verde"

## Resultado

Segurança virou propriedade do time de engenharia, não bottleneck trimestral.
